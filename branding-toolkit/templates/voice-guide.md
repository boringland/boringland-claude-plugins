### Tone by Context

| Context | Tone | Example |
|---------|------|---------|

| Success | Celebratory (restrained) | "Done! Your changes are live." |

## Writing Patterns

### Buttons & CTAs

**Do:**
- Start with action verb
- Be specific about outcome
- Keep under 3 words

**Examples:**
| ✅ Do | ❌ Don't |
|-------|---------|
| Create adventure | Submit |
| Start exploring | Click here |
| Save changes | Process |

### Error Messages

**Structure:** What happened + Why (if helpful) + What to do

**Examples:**
| ✅ Do | ❌ Don't |
|-------|---------|
| "Couldn't save. Check your connection and try again." | "Error 500" |
| "That name is already taken. Try a different one." | "Invalid input" |
| "We couldn't find that page. It may have been moved." | "404 Not Found" |

**Guidelines:**
- Never blame the user
- Avoid technical jargon
- Provide actionable next steps
- Use active voice

### Success Messages

**Examples:**
| ✅ Do | ❌ Don't |
|-------|---------|
| "Changes saved" | "Operation completed successfully" |
| "You're all set!" | "Success!" |
| "Adventure created! Add some characters." | "Record inserted into database" |

**Guidelines:**
- Keep brief
- Suggest next step when relevant
- Don't over-celebrate routine actions

### Empty States

**Structure:** What would be here + How to populate

**Examples:**
| ✅ Do | ❌ Don't |
|-------|---------|
| "No adventures yet. Create your first one." | "No data" |
| "Your inventory is empty. Pick up items as you explore." | "0 results" |
| "No messages. Start a conversation!" | "Nothing here" |

### Loading States

**Examples:**
| ✅ Do | ❌ Don't |
|-------|---------|
| "Creating your world..." | "Loading..." |
| "Almost there..." | "Please wait" |
| "Generating 12 scenes..." | "Processing" |

## Content Patterns

### Headlines

- Lead with benefit, not feature
- Use active voice
- Create curiosity

**Examples:**
| ✅ Do | ❌ Don't |
|-------|---------|
| "Bring your stories to life" | "AI-powered generation" |
| "Create worlds with words" | "Welcome to our platform" |

### Form Labels

- Be specific and descriptive
- Sentence case
- No trailing colons

**Examples:**
| ✅ Do | ❌ Don't |
|-------|---------|
| "Adventure name" | "Name:" |
| "Email address" | "EMAIL" |

### Placeholders

- Show format when relevant
- Don't repeat label
- Make deletable

**Examples:**
| ✅ Do | ❌ Don't |
|-------|---------|
| "the-lost-temple" | "Enter name here" |
| "you@example.com" | "Your email" |

## Before/After Examples

### Error Message

**Before:**
```
Error: Network request failed. Status code 500.
Please check your internet connection and try again.
```

**After:**
```
Couldn't reach our servers. Check your connection and try again.
```

### Confirmation Dialog

**Before:**
```
Title: Confirm Deletion
Body: Are you sure you want to delete this item?
Buttons: [OK] [Cancel]
```

**After:**
```
Title: Delete "The Lost Temple"?
Body: This adventure and all its scenes will be gone forever.
Buttons: [Delete] [Keep it]
```

### Welcome Message

**Before:**
```
Welcome to the application. Please complete your profile to continue.
```

**After:**
```
Let's set up your creator profile. This helps us personalize your adventures.
```
