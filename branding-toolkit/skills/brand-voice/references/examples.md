# Voice Examples: Before and After

## Error Messages

### Connection Error

**Before (Technical)**
```
Error: Network request failed. Status code 500.
Please check your internet connection and try again.
```

**After (Brand Voice)**
```
Couldn't reach our servers. Check your connection and try again.
```

**Why better:** Removes technical jargon (status code), uses "our servers" to take ownership, actionable.

---

### Validation Error

**Before (Blaming)**
```
You entered an invalid email address. Please enter a valid email.
```

**After (Brand Voice)**
```
That doesn't look like an email address. Try something like name@example.com
```

**Why better:** Doesn't blame user ("you entered"), gives helpful example.

---

### Permission Error

**Before (Robotic)**
```
Access denied. You do not have permission to perform this action.
```

**After (Brand Voice)**
```
You don't have access to this adventure. Ask the creator to invite you.
```

**Why better:** Explains what they can't access, provides actionable next step.

---

### Not Found

**Before (Generic)**
```
404 Error: Page not found
```

**After (Brand Voice)**
```
This adventure doesn't exist. It may have been deleted, or the link might be wrong.
```

**Why better:** Explains possible causes, removes error code.

---

## Confirmation Dialogs

### Delete Confirmation

**Before**
```
Title: Confirm Deletion
Body: Are you sure you want to delete this item?
This action cannot be undone.
Buttons: [OK] [Cancel]
```

**After**
```
Title: Delete "The Lost Temple"?
Body: This adventure and all its scenes will be gone forever.
Buttons: [Delete] [Keep it]
```

**Why better:** Specific about what's being deleted, descriptive buttons, conversational.

---

### Unsaved Changes

**Before**
```
Title: Warning
Body: You have unsaved changes. Do you want to save before leaving?
Buttons: [Yes] [No] [Cancel]
```

**After**
```
Title: Unsaved changes
Body: Leave without saving?
Buttons: [Save & leave] [Leave anyway] [Keep editing]
```

**Why better:** Removes "warning", descriptive buttons, clearer options.
